# Nikis Lawn & Garden Services

The website for **Nikis Lawn & Garden Services** — a friendly, Norwich-based
garden care team. Lawn care, hedge trimming, garden clearance, planting and
weekly maintenance.

## Tech

- Plain **HTML / CSS / JavaScript** — no frameworks, no build step.
- Hosted as a static site (works on GitHub Pages out of the box thanks to
  the `.nojekyll` file).

## Files

- `index.html` — page markup, semantic sections.
- `styles.css` — design tokens, layout, components, motion.
- `script.js` — carousel, accordion FAQ, before/after slider, animated
  stats, testimonials, search suggestions, form validation, back-to-top.
- `images/` — real before / after photos.

## Local development

Just open `index.html` in a browser, or serve the folder with any static
server, for example:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
